## 📌 Project Overview

This project is a modern and responsive website developed for an agricultural development cooperative.
The main objective was to create a clean, accessible, and informative digital presence that effectively represents the organization and its activities.

The website focuses on usability, simplicity, and structured content delivery for a wide audience.

---

## 🚀 Features

* Responsive and user-friendly design
* Clear navigation structure
* Informational content pages
* News/updates section
* Image gallery integration
* Contact page with essential details
